# 🧮 תלוי במספר (HangMath)

משחק דפדפן אינטראקטיבי ואלגנטי לתרגול פעולות חשבון לילדים. שחקו נגד השעון, ענו נכון, והצילו את האיש התלוי מפסילה!

## ✨ פיצ'רים מרכזיים
- **התאמה אישית**: בחירת פעולות לתרגול (חיבור, חיסור, כפל וחילוק) וקביעת תוצאת המקסימום.
- **דרגת קושי עולה**: ככל שמתקדמים בשלבים, הזמן מתקצר והתרגילים הופכים למאתגרים יותר.
- **שאלות בונוס**: מענה רצוף ונכון מזכה בשאלת בונוס שמאפשרת למחוק טעות מהאיש התלוי ולהרוויח ניקוד כפול!
- **עיצוב נקי ומינימליסטי**: ממשק מודרני, חלק, נעים לעין ורספונסיבי.

## 🛠️ טכנולוגיות
המשחק נבנה בצורה קלה ומהירה, כקובץ יחיד וללא תלויות חיצוניות (Frameworks):
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## 🚀 איך מריצים?
פשוט מורידים את הקובץ `index.html` ופותחים אותו בכל דפדפן רגיל, או נכנסים לגרסת הרשת דרך GitHub Pages (במידה ומוגדר).
